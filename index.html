Fundamentos

Diseño

Prompt Engineering como Diseño de Interfaces

Un prompt es una API call sin tipos. Estructura contract-first aplicada a lenguaje natural. Rol + contexto + restricciones + output. Few-shot, outputs estructurados (JSON, tablas), restricciones duras vs blandas.

Contract-FirstFew-ShotJSON OutputRestricciones
Conexión .NET Diseño de APIs → Diseño de prompts. El mismo mindset.
